RGB — Intensidades de Luz Aditiva

Tres fuentes de luz que se suman. La base de todo.

Modelo aditivo

RGB representa el color como la suma de tres intensidades de luz: Rojo, Verde y Azul. A mayor valor, más luz. Suma máxima = blanco.

¿Por qué 0–255? Cada canal ocupa 1 byte = 8 bits = 2⁸ = 256 estados (0..255). Tres canales = 24 bits = 16 777 216 colores.

HEX — Notación Hexadecimal de RGB

No es un modelo distinto. Es RGB escrito en base 16.

Sistema hexadecimal

Base 16: dígitos 0–9, A–F. Dos dígitos hex = 1 byte (16² = 256 valores). Perfecto para representar cada canal RGB.

#RRGGBB = rgb(R, G, B) — idénticos. HEX no añade información; sólo compacta la representación.

DEC → HEX:
  n = q·16 + r
  → q = primer dígito
  → r = segundo dígito

Ejemplos:
    0 → 00
   64 → 40
  128 → 80
  192 → C0
  255 → FF  (15·16 + 15)

HEX → DEC:
  FF = 15·16 + 15 = 255
  80 = 8·16 + 0 = 128
  A3 = 10·16 + 3 = 163

HSL — Doble Cono Bicónico

Hue · Saturation · Lightness en coordenadas cilíndricas

Geometría formal

HSL usa coordenadas cilíndricas formando un bicono: dos conos unidos en el ecuador (L = 50%).

Eje vertical = Lightness
L=0% → negro · L=50% → colores puros · L=100% → blanco
Radio: r(L) = 1 − |2L − 1|
Ángulo = Hue
0°=R · 60°=Y · 120°=G · 180°=C · 240°=B · 300°=M
Radio = Saturation
S=0% → eje (gris) · S=100% → borde (color puro)
⚠ El eje vertical NO es color. Los polos son siempre blanco/negro.
RGB → HSL
─────────────────────
R'=R/255  G'=G/255  B'=B/255
max=max(R',G',B')
min=min(R',G',B')
Δ = max − min

L = (max + min) / 2

S = 0              si Δ=0
  = Δ/(1−|2L−1|)  si Δ≠0

H = 60·((G'−B')/Δ mod 6)  si max=R'
  = 60·((B'−R')/Δ + 2)    si max=G'
  = 60·((R'−G')/Δ + 4)    si max=B'

HSL → RGB
─────────────────────
C = (1−|2L−1|)·S
X = C·(1−|(H/60) mod 2−1|)
m = L − C/2
(R,G,B) = (R'+m, G'+m, B'+m)·255

HSV — Cono Simple

Hue · Saturation · Value — el modelo natural de los selectores

Geometría formal

HSV (también HSB) tiene forma de cono simple. La punta inferior es negro (V=0). El disco superior muestra colores de máximo brillo.

Punta: V=0% → Negro
Disco superior (V=100%)
Centro S=0% → blanco · Borde S=100% → color puro
